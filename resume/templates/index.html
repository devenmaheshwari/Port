Projects

Methane Bubble Sound

A research project under the supervision of Professor Shima Abadi and the Ocean Data Lab. The focus of this project was to uncover the source of underwater methane plumes on the seafloor of the Puget Sound. As a researcher, I took data samples on the R/C Rachel Carson in the field and analyzed this data by generating power spectral density (PSD) graphs. We determined that the source of the methane vents were biological in nature.

3d Sustainable Materials Library

This personal project had sustainable design in mind as I attemped to create a materials library filled with sustainable elements. This software can be imported into any design interface and used to design furniture, clothing, or anything else that can be made with these materials.

Cookie Dropper Robot

A fun project where I built and programmed a robot that is meant to take a cookie, drop some chocolate syrup on it and return it back to the user via a conveyer belt (No food in demo!).
